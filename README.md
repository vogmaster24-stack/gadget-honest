# GadgetHonest - Affiliate Product Review Website

A minimal, modern affiliate website for honest product reviews.

## 🚀 Features
- Clean, mobile-friendly design
- Product review templates
- Blog layout
- Affiliate buttons & AdSense ready
- Easy manual updates

## 📁 Structure
- `index.html` - Homepage
- `product-review.html` - Individual review template
- `blog.html` - Blog listing
- `about.html` - About page
- `contact.html` - Contact form
- `privacy.html` - Privacy policy
- `affiliate-disclosure.html` - Disclosure

## 🎨 Design
- Primary: Dark Blue (#0a2647)
- Accent: Orange (#f97316)
- Font: Inter

## 🛠 Usage
1. Clone this repo
2. Update product images and text
3. Replace affiliate links
4. Deploy to GitHub Pages or any hosting
